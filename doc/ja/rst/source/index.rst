雑用はguippyにやらせよう
=========================
guippyはマウスを動かしたり、キーボードを叩いたりするためのソフトウェアです。あなたはキーボードやマウスの繰り返しの操作をしていたりしませんか？そういった作業はguippyにやらせて見ましょう。あなたはguippyに何を(例えばクリック)、どの順番で行うかをguippyに教えてやればいいのです。あとはguippyがあなたの指示通りにやってくれます。しかもそこそこ速くやってくれます。あなたは退屈な雑用から解放されて、大切な判断やクリエイティブなことに専念できるでしょう。
